# wings.js
A jQuery plugin that adds wings to a header

Usage:

Load the .js file as a script
```html
<script src="js/wings.min.js"></script>
```

Call the wings method on $(your object)
```html
<script>
	  jQuery(document).ready(function($) {
          
          $('#myHeading').wings();
          
        });
	</script>
	```
